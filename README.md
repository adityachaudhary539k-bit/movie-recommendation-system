# movie-recommendation-system
A content bases movie recommendation system engine built from scratch in python.the system analyses movie genres to calculate mathematical similarities between films,allowing users to input a movie and instantly recieve recieve a curated list of top recommendations
data processimg:-manages and structured the custom movie dataset using pandas
text vectorization:- implemented sickit-learn's TF-IDF vectorizer to transform text based genre tags into numerical vectors that the computer can analyze.
similarity engine:-utilized cosine similarity to calculate the mathematical distance between movie vectors,ensuring accurate,content driven recommenadations
user centric design:- engineered input handling to naturally account for accidental spacing and case-sensitivity,creating a smooth user experience.
tech stack used:- language-python, libraries-pandas,scikit-learn
