# 🎬 Movie Recommendation System

A content-based movie recommendation engine built from scratch in Python. The system analyzes movie genres to calculate mathematical similarities between films, allowing users to input a movie and instantly receive a curated list of top recommendations.

### 🚀 Key Features
* Data Processing: Cleaned and structured a custom movie dataset using Pandas.
* Text Vectorization: Implemented Scikit-Learn's TF-IDF Vectorizer to transform text-based genre tags into numerical matrices.
* Similarity Engine: Utilized Cosine Similarity to calculate the mathematical distance between movie vectors, ensuring accurate, content-driven recommendations.
* Robust Input Handling: Engineered input filtering to handle trailing spaces and case-sensitivity, preventing system crashes during user queries.

### 🛠️ Tech Stack
* Language: Python
* Libraries: Pandas, Scikit-Learn
